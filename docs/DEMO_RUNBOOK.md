# Demo runbook

## Before recording
1. `make quality`
2. `make live-preflight`
3. Confirm the escrow has enough MockUSDC.
4. Confirm the derived KeeperHub organization wallet has `EXECUTOR_ROLE` and testnet gas.
5. Clear Streamlit state and close unrelated tabs.

## 90-second path
1. Show funded milestone and 5% protected retention.
2. Execute the approved case.
3. Point to simulation success, KeeperHub execution id and explorer transaction.
4. Disable signed acceptance.
5. Execute again and show rejection before KeeperHub.
6. End on architecture and iArtisanat commercial continuation.

## Failure fallback
Never show a stub as live. If the network fails, show the previously verified explorer transaction and run the rejected path live, while stating exactly which step is prerecorded.
