# 90-second demo script

**0:00–0:12 — Problem**
“Construction clients fear paying too early; artisans suffer from late payments. Rook turns each milestone into an evidence-gated payment mandate.”

**0:12–0:28 — Product**
Show the funded amount, release amount, 5% retention, signed acceptance, invoice, compliance and no-dispute controls.

**0:28–0:48 — Decision**
Click **Evaluate policy**. Show deterministic approval and the exact releasable amount. State that an AI agent can recommend, but cannot bypass these controls.

**0:48–1:08 — KeeperHub execution**
Click **Execute through KeeperHub**. Show simulation success, execution id, transaction hash and open the explorer link. State that KeeperHub handles the last-mile execution with simulation and retry-safe idempotency.

**1:08–1:22 — Failure safety**
Uncheck client acceptance and execute. Show rejection before any transaction is attempted.

**1:22–1:30 — Business**
“After the hackathon, Rook becomes Secure Payments inside iArtisanat: subscription plus a fee per protected milestone.”
